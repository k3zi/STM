App Rundown

3rd Party Frameworks

These are the frameworks that we will use inside the iOS app. The app utilizes CocoaPods to install these frameworks. - AeroGearHttp - Handles GET/POST requests and uploading files to server - ObjectMapper - Converts server objects into their coresponding models - PureLayout - Allows easy manipulation of view layouts - Kingfisher - Lazy loads images into UIImageView - IQKeyboardManagerSwift - Automatically shifts text inputs so they appear above the keyboard - SecureNSUserDefaults - Used for storring user data securely - SpinKit - Displays a indicator on the screem - M13ProgressSuite - Displays progress bars and animations for loading

  • Folder/Developmemt structure will be MVC or Model–View–Controller

    • View - Updates itself depending on the data in the model
    • Controllers - encapsulates a main view (and it’s subviews) and displays it on the screen
